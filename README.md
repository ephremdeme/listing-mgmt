# Listing Management  

> Simple and minimal Listing Management Using NodeJs and React/Redux

## Features

- Basic CRUD
- Comment
- Rating
- Authentication 
- Elegant UI
- MySQL data store using Sequelize ORM

## Installation

```bash
git clone https://github.com/ephremdeme/listing-mgmt.git
cd listing-mgmt && npm install
npm run dev
```
